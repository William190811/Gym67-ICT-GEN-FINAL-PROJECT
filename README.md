# Gym67-ICT-GEN-FINAL-PROJECT
Gym67 offers high-quality equipment, apparel, and accessories to elevate every workout. From beginners to athletes, we provide durable gear, expert picks, and everything you need to train smarter, perform better, and reach your fitness goals with confidence.

**Tech Stack**

- HTML5 + CSS3 (pure, no frameworks)
- PHP 8
- MySQL (via XAMPP)
- XAMPP (Apache + MySQL local server)

---

 **Features**
 
- User registration and login with session management
- Dynamic product catalog fetched from MySQL database
- Category and price filters on shop page
- Individual product detail pages
- Customer reviews system
- Contact form with database storage
- Newsletter subscription
- Fully responsive layout
